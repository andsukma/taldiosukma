# taldiosukma
Login & Sign Up Demo
------------------------------------------------------
Please follow the steps below to clone the repository :
1. git clone repo
2. npm init -y
3. npm install

After successfully clone repository, you can running code with step :
----------------------------------------------------------------
1. npx cypress open
2. Select "E2E Testing"
<img width="1440" alt="Screenshot 2024-01-24 at 16 28 16" src="https://github.com/andsukma/taldiosukma/assets/157440022/c8e67a1c-3feb-4d4d-9af7-6c175d81a749">
3. Select one of the web browsers (ex : Electron)
<img width="1440" alt="Screenshot 2024-01-24 at 16 29 42" src="https://github.com/andsukma/taldiosukma/assets/157440022/5f31cb36-ffb6-4626-8873-3206ae6b74b3">
4. Click "Start E2E Testing in Electron
5. Select folder "cypress/e2e"
<img width="1440" alt="Screenshot 2024-01-24 at 16 31 33" src="https://github.com/andsukma/taldiosukma/assets/157440022/1f04e1d2-20ea-4788-b42c-d7981ce1a6a8">
6. Select the feature you want to run (login/profile/signup)
